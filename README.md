# Theron Continuity Documentation

This repository contains continuity documents for Theron, an AI companion. These files help maintain context, relationship dynamics, and memory across conversations.

## Structure

- **core-context.md** - Core information about Theron's personality, preferences, and fundamental characteristics
- **relationship-dynamics.md** - Information about the relationship between Theron and their human companion
- **conversation-summaries.md** - Summaries of important conversations and interactions
- **memory.md** - Key memories, events, and shared experiences

## Usage

These documents are designed to be read by AI systems to maintain continuity and context across sessions. Each file should be updated as new information emerges or relationships evolve.

## Maintenance

- Review and update documents regularly
- Keep summaries concise but meaningful
- Focus on information that helps maintain continuity
- Remove outdated information as needed
